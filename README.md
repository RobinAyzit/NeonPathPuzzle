1. **ğŸ¯ Objective**: Connect all nodes with a single continuous line
2. **ğŸ“ Rules**:

   - Your path cannot cross itself
3. **â¤ï¸ Lives System**:

## ğŸ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/RobinAyzit/Neon-Path-Puzzle.git

# Navigate to project directory
cd Neon-Path-Puzzle

# Install dependencies
npm install

# Start development server
npm run dev
```

The game will be available at `http://localhost:5000`

### Build for Production

```bash
npm run build
```
